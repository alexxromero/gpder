# gpder
### Gaussian processes with derivative observations

The gpder is a python package to implement gaussian processes with
derivative observations. The implementation is based on

“Derivative observations in Gaussian process models of dynamic systems” by
E. Solak, R. Murray-Smith, W. E. Leithead, D. J. Leith, and C. E. Rasmussen,
in Advances in Neural Information Processing Systems 15,
(Vancouver, British Columbia, Canada), 2002.

##### Installation

    pip install gpder
